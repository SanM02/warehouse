OPCION A (RECOMENDADA): Modificar angular.json para incluir src/assets
-------------------------------------------------------------------------

Archivo: cabravietnamirachamsinpeladofrontend/angular.json

UBICAR LINEAS 27-31 (seccion "assets"):

CODIGO ACTUAL:
            "assets": [
              {
                "glob": "**/*",
                "input": "public"
              }
            ],

CAMBIAR A:
            "assets": [
              {
                "glob": "**/*",
                "input": "public"
              },
              {
                "glob": "**/*",
                "input": "src/assets",
                "output": "/assets"
              }
            ],


OPCION B (ALTERNATIVA): Mover el favicon a public/
--------------------------------------------------
Si prefieres no modificar angular.json:
- Copiar src/assets/favicon.png a public/favicon.png
- Actualizar index.html para usar href="favicon.png" en vez de "assets/favicon.png"

NOTA: Recomiendo OPCION A porque es mas estandar y permite usar src/assets para
futuras imagenes sin problemas.
